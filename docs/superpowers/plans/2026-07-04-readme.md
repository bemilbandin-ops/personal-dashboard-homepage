- [x] **Step 3: Document the optional Windows launcher**

Use:

```powershell
powershell -ExecutionPolicy Bypass -File .\launcher-helper\install.ps1
```

Explain that installation is per-user, uses `HKCU`, requires Python 3 with `pythonw.exe`, launches only existing absolute `.exe` and `.lnk` targets, and exits after one launch. Warn readers never to commit `launcher-helper/secret.txt` or a generated token in `src/launcher-config.js`.

Use this uninstall command:

```powershell
powershell -ExecutionPolicy Bypass -File .\launcher-helper\uninstall.ps1
```

- [x] **Step 4: Document Chrome new-tab installation**

List the official unpacked-extension flow: open `chrome://extensions`, enable Developer mode, choose **Load unpacked**, and select the repository directory containing `manifest.json`. Note that Chrome must reload the extension after source changes.
